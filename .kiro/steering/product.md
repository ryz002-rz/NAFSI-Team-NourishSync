# Product Overview

NourishNet is a food access tool designed to connect communities with food resources. The application serves three primary user groups through dedicated portals:

- **Customer Portal**: Helps individuals find food pantries, meal programs, and food assistance locations with dietary filtering and health attributes
- **Donor Portal**: Connects organizations and individuals who want to donate food or resources with locations that need them
- **Volunteer Portal**: Matches volunteers with food assistance missions based on skills and language capabilities

## Core Features

- Multi-language support (English, Spanish, Chinese, French, Amharic, Tagalog)
- Interactive map visualization using Leaflet.js
- Dietary and health attribute filtering (Halal, Vegan, Vegetarian, No Beef, Low GI, Fresh Produce, Dairy Free)
- Voice search capability using Web Speech API
- Food insecurity index visualization
- Impact calculator for tracking carbon footprint and food waste diversion
